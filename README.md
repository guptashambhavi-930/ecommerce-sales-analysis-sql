# E-Commerce Sales Analysis using SQL

## Project Overview
This project analyzes e-commerce sales data using MySQL and SQL queries to generate business insights.

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Database Tables
### Customers
- Customer_ID
- Customer_Name
- Age
- Gender

### Orders
- Order_ID
- Customer_ID
- Product
- Category
- Sales
- Profit
- Region
- Order_Date

## SQL Analysis
- Total Sales Analysis
- Total Profit Analysis
- Top Selling Products
- Sales by Category
- Sales by Region
- Customer Order Analysis
- JOIN Operations

## Power BI Retail Sales Dashboard

### Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Category Analysis
- Sales by Region Analysis
- Monthly Sales Trend
- Interactive Region Filter

### Tools Used
- Power BI
- CSV Dataset
- Data Visualization
